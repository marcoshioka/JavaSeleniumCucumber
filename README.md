
## Setup

### Prerequisites

- Java 8 or higher
- Maven
- Google Chrome

### Installation

1. Clone the repository:
    ```sh
    git clone <REPOSITORY_URL>
    ```
2. Navigate to the project directory:
    ```sh
    cd <PROJECT_DIRECTORY>
    ```
3. Install Maven dependencies:
    ```sh
    mvn install
    ```

## Running Tests

To run the tests, use the following Maven command:
```sh
mvn test
